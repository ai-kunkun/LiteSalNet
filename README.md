# LiteSalNet
A PyTorch implementation of LiteSalNet for Remote Sensing Salient Object Detection.

# Network Architecture
![LiteSalNet Architecture](https://github.com/ai-kunkun/LiteSalNet/blob/main/image/LiteSalNet.png)

# Requirements
- Python 3.7
- PyTorch 1.9.0
